# 深度清理执行版

## 户型区域对应
- 上方左侧：客厅2
- 上方右侧：主卧
- 中间左侧：客厅1
- 中间右侧：卫生间
- 下方左侧：厨房
- 下方右侧：次卧

## 执行顺序
- 先开窗通风，启动第一轮洗衣，晴天先晒被子和床品
- 先整理收纳和擦灰，再人工扫地、刷地、拖地
- 人工清洁完成后，再用扫地机器人补扫补拖
- 机器人运行时，同步做洗衣晾晒、台面清洁和工具整理

## 客厅区域

### 客厅2
- 整理杂物并清空台面
- 擦桌面、柜面、窗台、门把手和开关
- 清角落灰尘
- 人工扫地或吸尘
- 人工拖地

### 客厅1
- 收纳茶几、沙发表面物品
- 擦茶几、电视柜、窗台和门把手
- 移动沙发和茶几
- 清理沙发底部、茶几底部、家具背后、墙角和踢脚线
- 人工扫地或吸尘
- 人工拖地
- 家具复位

### 并行任务
- 洗衣机运行时继续整理收纳
- 衣物洗完立即晾晒
- 地面半干后启动扫地机器人清扫客厅

## 卧室区域

### 主卧
- 更换床单、被套、枕套并立即送洗
- 整理床铺和零散衣物
- 清空并擦拭床头柜、柜面、窗台、门把手和开关
- 挪开小家具，清理床边缝隙、床底和柜子侧边
- 人工扫地或吸尘
- 人工拖地
- 家具复位

### 次卧
- 整理床铺和杂物
- 清空并擦拭桌面、柜面、窗台和门把手
- 挪开轻便家具，清理角落和家具底部
- 人工扫地或吸尘
- 人工拖地
- 物品归位

### 并行任务
- 晒被子或翻面晾晒寝具
- 洗衣机运行时整理衣柜和叠衣服
- 地面半干后启动扫地机器人补扫卧室

## 厨房区域
- 清空台面，整理调料、瓶罐和杂物
- 倒垃圾
- 擦灶台、台面和水槽
- 清理重油污、墙角和地面边缘
- 挪开可移动物品后清理底部
- 人工扫地或吸尘
- 人工拖地

### 并行任务
- 洗衣机运行时先做灶台、台面、水槽
- 地面半干后启动扫地机器人补扫可达区域
- 机器人运行时清洗抹布和整理清洁用品

## 卫生间区域
- 整理洗漱用品并清空台面
- 擦镜子
- 清洁洗手台、水龙头和马桶
- 清理地漏、墙角和潮湿死角
- 刷洗地面
- 拖干地面或保持通风晾干

### 并行任务
- 毛巾、浴巾、抹布单独机洗
- 洗衣机运行时先做镜子、洗手台和马桶
- 外部可达区域交给扫地机器人补扫补拖

## 最后收尾
- 全屋检查一遍是否有遗漏
- 带出垃圾
- 清洗抹布、拖把和刷子
- 清理扫地机器人尘盒、滚刷和拖布
- 收回并整理晾晒好的衣物和被子

## 工具
- 垃圾袋
- 抹布
- 扫把和簸箕
- 拖把
- 吸尘器或除尘工具
- 厨房清洁剂
- 卫生间清洁剂
- 手套
- 小凳子
- 洗衣液或洗衣凝珠
- 晾衣架和夹子
